Introdução à programação web

O curso de Introdução à Programação Web é ideal para quem deseja aprender os conceitos básicos de desenvolvimento de sites e aplicações para a web. Durante o curso, os alunos aprenderão HTML, CSS e JavaScript, além de técnicas essenciais para criar páginas interativas e responsivas, com foco no design moderno e na experiência do usuário.

Algoritmo e estrutura de dados

O curso de Algoritmo e Estrutura de Dados oferece uma introdução sólida ao mundo da programação de computadores. Os alunos aprenderão como resolver problemas computacionais através da implementação de algoritmos eficientes e estruturas de dados, como listas, pilhas, filas e árvores, preparando-os para desafios mais complexos em ciência da computação.

Grafos

O curso de Grafos é focado no estudo das representações gráficas de redes e suas aplicações em áreas como redes sociais, rotas de transporte e sistemas de recomendação. Os estudantes aprenderão os fundamentos de grafos, suas propriedades e algoritmos de busca, além de como aplicar esses conceitos na resolução de problemas do mundo real.
